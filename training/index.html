Tutorial


Seccion 1: Configurando los programas principales

  1. Crear una cuenta de GitHub en caso de no tener una aquí.
  2. Instalar Github Desktop.
  3. Instalar Godot
  4. Clonar el repositorio del juego. (desde Github Desktop)
  5. Arrastrar la carpeta del repositorio a Godot (esto va a agregar el proyecto a Godot)


Seccion 2: Configurando Trello

  1. Crear una cuenta de Trello, en caso de no tener una aquí.
  2. Unirse a la tabla.


Dudas

  • Como subir un cambio del juego: Ir a Github Desktop, y abajo a la izquierda, escribir el nombre y la descripcion del cambio, luego, apretar "Commit to main" y después en "Push origin".
